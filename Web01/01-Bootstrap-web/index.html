Tlačítka

bootstrap umožňuje snadné stylování tlačítek pomocí tříd jako například např. btn-primary

 
    <button class="btn btn-primary"> Primární < /button>
    <button class="btn btn-warning"> Warning button </button>

Karty

Tabulky

Tabulky lze v Bootstrapu stylovat pomocí tříd jako např. table-striped

# Jméno e-mail
1 Jan novák jannovak@e-mail.cz
2 Jana Nováková jana.novakova@e-mail.cz
3 Petr novák petr.ovak@e-mail.cz 

    <table class="table table-striped  table-hover">
        <thead class="table-dark">
         <tr>
            <th>#</th>
            <th> Jméno</th>
            <th> e-mail</th>
         </tr>
        </thead>
    <tbody>
    <tr>
        <td>1</td>
        <td> Jan novák</td>
        <td> jannovak@e-mail.cz</td>
    </tr>
    <tr>
        <td>2</td>
        <td> Jana Nováková</td>
        <td> jana.novakova@e-mail.cz</td>
    </tr>
    <tr>
        <td>3</td>
        <td> Petr novák</td>
        <td> petr.ovak@e-mail.cz</td>
    </tr> 
    </tbody>
    </table>

Kontrola

Kontrola v Bootstrapu jde vytvářet jako zaškrtávaící políčka pomocí form-check

 
    <div class="form-check">
        <input class="form-check-input" type="checkbox" value="" id="flexCheckDefault">
        <label class="form-check-label" for="flexCheckDefault">
          Obyčejné políčko
        </label>
      </div>
      <div class="form-check">
        <input class="form-check-input" type="checkbox" value="" id="flexCheckChecked" checked>
        <label class="form-check-label" for="flexCheckChecked">
          Zaškrtlé políčko
        </label>
      </div>


      <div class="form-check mt-4">
        <input class="form-check-input" type="radio" name="flexRadioDefault" id="flexRadioDefault1">
        <label class="form-check-label" for="flexRadioDefault1">
          buď
        </label>
      </div>
      <div class="form-check">
        <input class="form-check-input" type="radio" name="flexRadioDefault" id="flexRadioDefault2" checked>
        <label class="form-check-label" for="flexRadioDefault2">
          nebo
        </label>
      </div>

Výběr

        <select class="form-select" aria-label="Default select example">
            <option selected>otevřít menu</option>
            <option value="1">možnost jedna</option>
            <option value="2">možnost dva</option>
            <option value="3">možnost tři</option>
          </select>
